# Resume Reviewer - using Gen AI

An AI-powered web application that helps candidates prepare for interviews by analyzing their resume and job description to <b>generate technical questions, behavioral questions, skill gaps, and a personalized roadmap.</b>

# Tech Stack
- Node.js
- Express.js
- React.js
- MongoDB

# AI Integration
- Gemini AI API
